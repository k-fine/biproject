사내와 클라우드에서 사용자 정의, 패키지 및 메인프레임 애플리케이션과 데이터베이스를 상호 운용할 수 있도록 하고

특징

간편한 확장성

Software AG의 개방형 통합 플랫폼은 쉽게 확장 가능하며, 사내 애플리케이션을 SaaS 애플리케이션에 통합할 수 있으며, 주요 기업에서 검증 가능 분석가들은 통합 기능에 대해 Software AG를 지속적으로 인식

300개 이상의 커넥터

수십 개의 커넥터를 통해 응용 프로그램을 웹 메서드 통합 서버에 연결 가능 코딩 불필요 이러한 커넥터는 Salesforce.com®, SAP, Oracle®, J2EE®, .NET® 및 기타 대부분의 선도적인 기술

더 빠른 통합

시장 출시 기간 단축 웹 메서드 통합 서버를 통해 새로운 기회와 고객의 요구에 신속하게 대응

"Lift & shift" 통합

"Lift & Shift"는 클라우드로 신속하게 전환할 수 있는 클라우드 도입 전략 웹 메서드 통합을 클라우드로 쉽게 "이동＂ 가능 추가 하드웨어 비용이나 설치 없이 가치 실현 시간을 단축

매핑 및 변환

고유한 응용 프로그램을 통합하려면 각 응용 프로그램 형식에 대한 데이터 요소를 매핑하고 연결 변환 서비스 라이브러리와 그래픽 매핑 UI를 사용하여 데이터 형식을 서로 연결

클라우드 내 iPaaS 통합

NAT의 통합 서비스형 플랫폼(iPaaS)과 클라우드 내 애플리케이션을 신속하게 통합 비기술적 비즈니스 담당자는 통합을 주도할 수 있으므로 IT가 보다 복잡한 과제에 집중할 수 있는 시간을 더 많이 확보 가능
