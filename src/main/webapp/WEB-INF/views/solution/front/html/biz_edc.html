Informatica EDC(Enterprise Data Catalogue)

Informatica EDC(Enterprise Data Catalogue)는 기업에 메타데이터 기록 시스템을 제공할 수 있는 기계 학습 기반 데이터 카탈로그입니다.

  • Informatica EDC
  • Works
모든 데이터 자산의 연결 및 카탈로그화

모든 환경에서 데이터 자산을 분류 및 구성하여 데이터 가치와 재사용을 극대화하고,
기업에 메타데이터 기록 시스템을 제공할 수 있는 기계 학습 기반 데이터 카탈로그입니다.

특징

모든 데이터 자산의 연결 및 카탈로그화

자동 스캔으로 온-프레미스, 클라우드, 빅 데이터 플랫폼에서뿐만 아니라, BI 도구, ETL, 타사 메타데이터 카탈로그에서, 그리고 구조화된 데이터 형식과 구조화되지 않은 데이터 형식에서 자산을 검색하고 카탈로그화

데이터 프로비저닝

간단한 클릭 연결 데이터 프로비저닝으로 더 많은 셀프 서비스를 지원하여 원하는 대상에 데이터를 제공

엔드 투 엔드 데이터 계보

높은 수준의 시스템 보기에서 세분화된 열/메트릭 수준 계보 및 상세한 영향 분석에 이르기까지 데이터 이동을 완벽하게 추적 가능

데이터 관계 및 권장 사항

연결된 표, 뷰, 도메인, 보고서를 신속히 파악 컬럼 유사성 및 추론 도메인을 기반으로 한 인텔리전트 데이터 권장 사항을 참조

AI 기반 자동화

AI 기반 도메인 검색, 데이터 유사성, 비즈니스 용어 연결 및 권장 사항을 통해 데이터 큐레이션을 자동화

협업

인증, 등급 및 리뷰, Q&A 플랫폼 및 변경 알림으로 공유 데이터 지식을 활용 가능

Tableau 확장

기본 Tableau 사용자 인터페이스 내에 있는 계보, 인증, 동료 리뷰 및 지능형 메타데이터를 기준으로 상황에 맞는 데이터를 검색 가능

장점

데이터 품질 향상

기업 내에서 중요하고 신뢰할 수 있는 데이터의 위치를​ 검색하는 데 도움
의미, 사용 및 비즈니스 컨텍스트에 대한 명확한 정보를 얻을 수 있으므로 데이터 품질 향상

데이터 통합​노력 지원

비즈니스 컨텍스트와 함께 데이터 가시성을 제공하여 비즈니스-IT 협업을 촉진하고, 민첩한 프로토타이핑 및 데이터 통합 파이프라인 개발 용이

마스터 데이터 식별 및 분류

메타데이터 기반 프로파일링은 데이터의 정확성과 완전성에 대한 통찰력을 제공하므로 MDM 이니셔티브를 보다 쉽게 계획하고 셀프서비스를 지원

데이터 거버넌스 강화

Enterprise Data Catalog 및 AI 기반 통찰력은 데이터 검색 및 카탈로그화 프로세스를 자동화하여 계속해서 증가하는 데이터의 양 정리 가능
