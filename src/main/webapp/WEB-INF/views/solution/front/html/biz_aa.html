특징

마케팅의 기반이 되는 웹 분석

수상을 통해 우수성이 입증된 툴을 사용 방대한 양의 웹 데이터를 실질적인 인사이트로 전환할 수 있도록 도움

마케팅 분석의 다양한 기능

Adobe는 모든 채널의 데이터를 수집 수집한 데이터를 한곳으로 통합하여 고객을 종합적으로 파악 실시간 인사이트를 제공

강력한 기여도 분석

Adobe가 더 나은 의사결정을 내리도록 도움 강력한 기여도 분석을 통해 전환율을 살펴보고, 그에 따라 투자 결정 가능

효율적인 예측 분석

Adobe가 복잡한 작업을 수월하게 할 수 있도록 도움 제공 인공 지능과 머신 러닝을 사용하여 한 번의 클릭만으로 잠재 기회를 확보하고, 문제에 대한 해답을 찾을 수 있음

주요 기능

분산된 데이터 통합

웹, 모바일, CRM, 커넥티드 자동차 등의 데이터를 통합하여 고객과 비즈니스의 전체 상황을 명확하게 파악

컨텍스트에 맞는 고객 경험 제공

데이터를 활용하여 고객의 행동과 그 이유 및 방식에 대한 인사이트를 도출함

신속하게 인사이트 확보
