한 화면에서 전체 고객 여정 파악

Adobe Campaign에서 풍부한 고객 데이터를 사용하여 이메일, 모바일, 오프라인 채널 등을 통해
고객이 실제로 원하는 다이내믹한 캠페인을 생성 및 조정하고 제공할 수 있습니다.

특징

이메일 마케팅

우수한 크로스채널 캠페인은 이메일이 기반

고객 여정

전체 여정을 확인하여 고객 자체 뿐만 아니라 고객이 브랜드에 도달하는 방식을 파악 가능

옴니채널 마케팅

언제 어디에서나 고객과 교류할 수 있는 크로스채널 캠페인을 구축 가능

주요 기능

이메일 개인화

개인화된 메시지를 통해 이메일 마케팅을 향상
Adobe Campaign의 기본적인 이메일 기능을 사용

캠페인 관리

시각적 워크플로우 인터페이스를 통해 여러 데이터 소스를 관리하거나 고객 세그먼트를 정의하고,
크로스채널 캠페인을 단계별로 계획하고 실행할 수 있음

프로모션 관리

규칙 기반의 프로모션 카탈로그 생성 가능
가중치나 우선 순위로 프로모션을 관리, 고객이 특정 프로모션을 보는 횟수를 제한

고객 세분화

다차원 타겟팅을 통해 고객 프로파일의 거의 모든 변수를 기반으로 세그먼트를 만들 수 있음
