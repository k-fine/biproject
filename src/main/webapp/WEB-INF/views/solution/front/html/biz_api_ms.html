API Monitoring System

통합적인 EAI 서비스 모니터링 시스템 구축을 통하여 장애 대응 및 EAI 서비스 관리의 효율성을 높이고,
예산낭비 및 업무 손실을 줄이기 위한 솔루션입니다.

특징

통합 모니터링 솔루션

일일 수십,수백만건의 Transaction을 실시간으로 모니터링 예외 상황 발생시 즉시 조치할 수 있도록 지원

현업 담당자의 참여 보장

업무시스템 담당자, IT 관리자 및 IDC 담당자 모두 활용 가능 담당자 별 실시간으로 오류를 확인하여 복구시간 단축

업무 중심 모니터링

현업 업무 중심으로 관리가 가능하도록 물리적, 논리적 구성 IT 담당자와 현업 간의 원활한 의사소통이 이루어지도록 지원

주요 기능

DASHBOARD

실시간 처리현황 감시 시간, 일, 월별에 따른 데이터 추이 분석 업무 그룹별 통계 서버 / 서비스 현황

서비스 모니터링

인터페이스 별 처리시간 및 결과확인 예외상황 발생 시 데이터 및 오류내용 즉시 확인 다양한 조건에 의한 검색 Transaction 오류 감지 예외 상황 발생 시 e-Mail 및 SMS 자동전송

시스템 관리

논리적인 분류체계를 통한 업무그룹관리 물리적인 시스템등록 및 논리모델과 연계관리 서비스 별 담당자 지정 및 모니터링 권한부여 업무중심의 서비스등록 및 관리

서비스 관리

전사 인터페이스 업무를 비즈니스 단위로 등록관리 N:M관계를 표현할 수 있는 인터페이스는 다양한 Source, Target간의 원활한 구성관리 가능 인터페이스에 관련된 담당자 및 산출물을 등록하여 향후 업무진행에 Help기능 제공
