적절한 활용 사례를 위한 적절한 구축 모델

데이터 마이그레이션에서 차세대 분석에 이르기까지 조직에서 수행 중인 이니셔티브 유형에 상관없이
모든 활용 사례에 대해 손쉽게 데이터 품질을 구축하는데 필요한 유연성을 제공합니다.

특징

적절한 워크로드를 위한 적절한 규모

팀, 일선 부서 전체 또는 IT 부서가 실시간의 웹 서비스, 배치 및 빅데이터를 위해 모든 워크로드에서
데이터 품질을 쉽게 구축할 수 있도록 도와 줌

적절한 개인을 위한 적절한 사용자 환경

팀의 각 구성원에게 최상의 사용자 환경을 제공하도록 설계 조직 전체에서 다양한 기능, 기술 역량 및 관심사를 충분히 처리할 수 있을 만큼 유연함

보다 우수한 생산성을 위한 적절한 자동화

지능형 권장 사항 및 평가 수행 가능 팀 구성원은 품질 통찰력을 사용하여 데이터 검색과 같은 중요한 작업을 자동화하여 생산성과 효율성 제고 가능

주요기능

역할 기반 기능

비즈니스 사용자의 역량을 강화하고 IT 및 비즈니스 이해 관계자 간 협업을 촉진

유연한 구축

모든 활용 사례 및 모든 워크로드에 대해 멀티 클라우드 및 기업 내 데이터의 품질을 관리

예외 관리

수동 작업을 워크플로에 통합하여 비즈니스 사용자가 자동화된 프로세스 전체에서 예외를 검토, 수정 및 승인 가능

Intelligent Data Quality

AI 중심의 통찰력을 사용하여 중요한 작업을 자동화하고 데이터 검색을 간소화하여 생산성과 효율성 제고

풍부한 데이터 품질 변환 세트

데이터 표준화, 유효성 검사, 보강, 중복 제거 및 통합 기능으로 고품질 정보를 제공
